## 📖 Overview

**Country Connection** is a community-led, land-based wellness retreat program for Aboriginal youth aged 13-18. The program combines a voluntary **5-day digital detox** — a complete break from phones, social media, and screens — with immersive cultural learning led by Elders and Knowledge Keepers.

This repository contains the complete business plan, operational templates, grant applications, presentation materials, and all forms needed to launch and run the program.

**Core philosophy:** You cannot take something away without putting something better in its place. We replace digital distraction with connection to Culture, Country, and Community.

---

## 🎯 Mission

To empower Aboriginal youth by using cultural connection as the foundation for digital wellness, fostering resilience, identity, and community through land-based retreats led by Elders.

We welcome contributions from community members, Elders, youth workers, grant writers, and anyone who shares our mission.

### Ways You Can Help

| Contribution Type | Description |
| :--- | :--- |
| **Cultural Guidance** | Review and improve cultural protocols, activity descriptions, and Elder engagement |
| **Language Support** | Add translations of forms and key documents (French, Cree, Ojibwe, etc.) |

### How to Submit a Contribution

1. **Fork this repository** (click the Fork button at top right)
2. **Create a new branch** for your changes
3. **Make your edits** (add files, update text, fix errors)
4. **Submit a Pull Request** with a clear description of what you changed and why

No GitHub experience? No problem. You can also:

- **Open an Issue** describing your suggestion
- **Email** the project maintainer directly
- **Share documents** via Google Drive and send the link

## 📋 Contribution Guidelines

### For Cultural Contributors

- Respect community protocols. All cultural content must be reviewed by Elders.
- Do not include sacred or restricted knowledge in public documents.
- Suggest additions or changes by opening an Issue first.

### For Grant Writers & Fundraisers

- Check existing grant sources before adding duplicates.
- Include deadlines, amounts, and eligibility requirements.
- Note whether the grant requires registered non-profit status.

### For Translators

- Create a new file with `_fr` (French), `_cr` (Cree), etc. suffix.
- Include translator name and date.
- Note any cultural adaptations made.

### For Developers & Designers

- Keep the HTML self-contained (no external dependencies).
- Maintain print-friendly CSS (test with `@media print`).
- Preserve accessibility (clear headings, readable fonts, high contrast).

## 💰 Financial Model

| Item | Amount |
| :--- | :--- |
| Cost per retreat (15 youth, 5 days) | $22,390 |
| Cost per youth | ~$1,400 |
| Annual goal (4 retreats) | $89,560 |
| Request to Band Council | **$0** |

**Funding sources:** Government grants (50-70%), foundations (20-30%), corporate sponsorships (10-20%), in-kind support.

**Youth participants pay nothing.**
